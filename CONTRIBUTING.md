# Contributing to WG Commander

## Issues

Bug reports, feature requests, and questions are welcome via GitHub Issues.
Typo reports are fine as issues too.

## Pull Requests

Before submitting a PR, please open an issue first to discuss the change.
This helps avoid wasted effort on both sides.

Please do not submit PRs for typo fixes, formatting, or cosmetic changes.
If you spot a typo, open an issue instead.

## AI Disclosure

If your contribution was written with AI assistance, please mention which tool you used and to what extent (e.g. "code generated by codex with GPT-8 with agi-xhigh" or "used Copilot for autocomplete"). This helps maintainers review with the right expectations.

## Building

```sh
make          # build for current platform
make release  # build linux amd64 + arm64
make test     # run tests
```
